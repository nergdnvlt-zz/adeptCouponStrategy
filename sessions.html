Demo for Sessions API

This demo is to show one way to allow a single coupon code to discount different products by different amounts using the /sessions API.

Key Points

  • Must collect info to create and account
  • An Account must be passed to the session
  • The session will then return a session ID
  • Pass the session ID to the checkout
